Expresiones humanas:

Cuando se seleccionan opciones predefinidas, el input muestra expresiones legibles como:

  • now - now para "Hoy"
  • now-1d - now-1d para "Ayer"
  • now-7d - now para "Últimos 7 días"
  • now-30d - now para "Últimos 30 días"
