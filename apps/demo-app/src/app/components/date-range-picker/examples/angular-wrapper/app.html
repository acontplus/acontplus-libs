Opciones de Configuración

Propiedades Principales:
  • [singleDatePicker] - Modo de fecha única
  • [autoApply] - Aplicar automáticamente la selección
  • [timePicker] - Habilitar selector de tiempo
  • [ranges] - Rangos predefinidos
  • [presetTheme] - Tema predefinido ('default', 'bootstrap', 'material')
  • [minDate] / [maxDate] - Fechas límite
  • [locale] - Configuración de idioma
Eventos:
  • (dateRangeSelected) - Cuando se selecciona un rango
  • (pickerShow) - Cuando se muestra el picker
  • (pickerHide) - Cuando se oculta el picker
  • (pickerApply) - Cuando se aplica la selección
  • (pickerCancel) - Cuando se cancela la selección
