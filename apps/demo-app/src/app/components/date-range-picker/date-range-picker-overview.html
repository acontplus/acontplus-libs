El NgDateRangePickerComponent es un componente Angular que envuelve la potente librería datex-ui para selección de rangos de fechas. Proporciona integración completa con Angular Forms, soporte para temas personalizables, selección de fechas individuales, rangos de fechas, selector de tiempo, rangos predefinidos, localización y muchas otras funciones avanzadas.
